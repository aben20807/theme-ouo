## Install and Set theme

### [Oh My Fish!](https://github.com/oh-my-fish)

```fish
$ omf install https://github.com/aben20807/theme-ouo
$ omf theme ouo
```

### [How to set fish as default shell in Cygwin](https://stackoverflow.com/a/34683320/6734174)

## Update

### [Oh My Fish!](https://github.com/oh-my-fish)

```fish
$ omf u ouo
```
